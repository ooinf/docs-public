# FAQ

## O que é o OOInfo?

É uma plataforma para criar listas estruturadas, colaborativas e pesquisáveis.

## Qual a diferença entre lista e item?

Lista é o conjunto. Item é cada registro dentro dela.

## Qual a diferença entre campo e tag?

Campo é parte fixa da estrutura da lista. Tag é uma palavra-chave flexível usada para classificação.

Use campo quando a informação aparece em todos ou quase todos os itens. Use tag quando a classificação pode variar bastante.

## Quando devo usar relação?

Use relação quando uma informação pertence a outra lista ou será reutilizada em vários lugares.

Exemplo: em vez de escrever o nome do autor em cada livro, crie uma lista de autores e relacione cada livro ao autor correto.

## Posso importar dados de planilha?

Sim. Use CSV para importar vários itens de uma vez.

Antes de enviar, confira cabeçalhos, datas, links e campos obrigatórios.

## Uma contribuição aparece imediatamente?

Depende das regras da lista. Algumas publicam direto; outras exigem revisão.

## Como sei qual visualização usar?

Use tabela para comparação, cards para descoberta, galeria para imagens, documento para leitura contínua e hierarquia para estruturas em árvore.

## Posso publicar uma lista em um site?

Sim. Você pode montar sites com listas e páginas, ou usar embed para exibir uma lista em outro endereço.

## Como evito expor dados que não deveriam ser públicos?

Revise a visibilidade da lista, permissões de colaboração, campos sensíveis e links compartilhados antes de publicar.

## Por que meus dados não aparecem na busca?

Verifique se o item foi salvo, se a lista permite visualização para o seu perfil e se os termos pesquisados aparecem em campos preenchidos.

## Posso mudar campos depois?

Sim, mas mudanças estruturais afetam preenchimento, filtros e importações futuras. Em listas grandes, planeje e teste antes de alterar campos essenciais.

## Como reduzir erros de contribuição?

Use nomes de campos claros, exemplos de preenchimento, campos obrigatórios com moderação e revisão quando a lista for aberta à participação.

# Documentação Pública do OOInfo

O OOInfo ajuda comunidades e equipes a transformar informações soltas em listas estruturadas, pesquisáveis e colaborativas.

Esta documentação é voltada para quem usa a plataforma no dia a dia: criar listas, preencher dados, colaborar, revisar contribuições, importar CSVs, publicar sites e encontrar informações.

## Modelo mental

Uma lista no OOInfo funciona como uma base de conhecimento com estrutura própria.

| Conceito | O que significa |
| --- | --- |
| Lista | Um conjunto de dados sobre um tema, como instituições, livros, projetos ou contatos. |
| Campo | Uma coluna da lista, com tipo definido: texto, número, data, imagem, relação e outros. |
| Item | Um registro dentro da lista. |
| Relação | Uma conexão entre itens de listas diferentes. |
| Visualização | A forma de ler a lista: tabela, cards, galeria, documento, hierarquia ou mapa. |

## Fluxo básico

1. Escolha o assunto da lista.
2. Defina quais campos cada item precisa ter.
3. Cadastre itens manualmente ou por CSV.
4. Ajuste visualização, filtros e permissões.
5. Convide colaboradores ou publique a lista em um site.

> Esta área contém apenas conteúdo público de uso comum. Materiais internos de operação e governança ficam fora deste site.

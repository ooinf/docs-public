# Busca e Permissões

Busca e permissões trabalham juntas: a busca ajuda a encontrar dados, e permissões definem quem pode ver ou modificar esses dados.

## Busca

Use busca para localizar texto livre em listas e conteúdos.

Dicas:

- pesquise por nomes, palavras-chave e trechos curtos;
- combine busca com filtros para refinar resultados;
- use filtros quando souber exatamente o campo que quer consultar.

## Melhorar resultados de busca

Resultados melhores começam na forma como a lista é preenchida.

- Use títulos claros.
- Preencha descrições curtas.
- Evite abreviações sem contexto.
- Use tags consistentes.
- Separe informações importantes em campos próprios.
- Mantenha relações corretas.

## Filtros

Filtros respondem perguntas estruturadas, como:

- itens de uma categoria;
- registros de uma cidade;
- materiais publicados em um período;
- itens com determinada tag;
- registros relacionados a outra lista.

## Busca ou filtro?

| Use busca quando | Use filtro quando |
| --- | --- |
| Você lembra uma palavra ou parte do nome. | Você sabe o campo exato. |
| Quer explorar resultados aproximados. | Quer restringir por categoria, data, tag ou relação. |
| O texto pode estar em vários campos. | O valor precisa ser preciso. |

## Permissões

Permissões controlam ações. Uma pessoa pode ter permissão para ver, editar, comentar, revisar ou gerenciar uma lista, dependendo das regras definidas pelo dono da lista.

## Tipos de acesso

| Acesso | O que permite |
| --- | --- |
| Visualizar | Ler dados e navegar pela lista. |
| Comentar | Participar de discussões quando comentários estão ativos. |
| Contribuir | Enviar novos itens ou mudanças conforme regras da lista. |
| Revisar | Avaliar contribuições pendentes. |
| Gerenciar | Ajustar estrutura, campos, visualizações e regras da lista. |

## Recomendações

- Comece com acesso mais restrito e amplie quando necessário.
- Revise permissões antes de publicar links.
- Use revisão para listas abertas à contribuição.
- Evite publicar dados pessoais sem base adequada e consentimento quando necessário.

## Antes de compartilhar um link

Confira:

- se visitantes conseguem abrir a lista;
- se pessoas sem permissão não conseguem editar;
- se itens pendentes não aparecem antes da revisão;
- se campos sensíveis não foram publicados por engano;
- se filtros e busca ajudam a encontrar o conteúdo principal.
